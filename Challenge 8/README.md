##Project Overview
This project analyzes national university ranking data using SQL, SQLite, and Python to uncover insights related to tuition trends, enrollment distribution, institutional rankings, and fee structures.

The analysis focuses on:
tuition revenue patterns,
enrollment distribution,
ranking-based financial comparisons,
and institutional performance analysis.

##Business Problem
Educational analysts and university stakeholders need to understand how tuition, enrollment, and rankings interact in order to:
evaluate institutional financial performance,
compare top-ranked and lower-ranked universities,
analyze enrollment concentration,
and identify trends in higher education costs.

This project uses university ranking data to generate business-oriented educational insights.

##Analysis Performed
The project explored:
in-state fee availability across universities,
enrollment distribution by fee category,
tuition revenue comparisons between top-ranked and lower-ranked universities,
schools generating the highest and lowest tuition totals,
and cumulative student enrollment among top-ranked institutions.

Analytical Thinking Demonstrated
Standardized inconsistent financial and enrollment data formats for analysis.
Applied ranking functions to compare institutional financial performance.
Used recursive SQL queries to calculate cumulative enrollment trends.
Combined tuition and enrollment metrics to estimate institutional fee generation.
Segmented universities based on in-state fee availability for comparative analysis.

##Key Insights
Universities with in-state fee structures represented a significant share of total student enrollment.
Top-ranked universities generated substantially higher estimated tuition revenue compared to lower-ranked institutions.
Enrollment concentration varied significantly across institutions.
Tuition and enrollment combined provided stronger insight into institutional financial scale than rankings alone.
Recursive enrollment analysis highlighted the concentration of students within top-ranked universities.

##Data Visualization
A bar chart was created using Matplotlib to compare universities with the highest and lowest total tuition fee generation.
The visualization highlighted major financial differences across institutions.
