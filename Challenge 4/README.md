##Project Overview
This project analyzes customer behavior data using SQL, SQLite, and Python to uncover insights related to customer satisfaction, purchasing behavior, loyalty performance, and promotional impact.

The analysis focuses on identifying:
customers at churn risk,
the effectiveness of promotions,
high-value customer segments,
and spending patterns across loyalty groups.

##Business Problem
Businesses need to better understand customer purchasing behavior and loyalty trends in order to:
improve retention,
optimize promotions,
identify high-value customers,
and increase revenue.

This project uses transactional and demographic data to support data-driven customer analytics and marketing decisions.

##Key Insights
Customers with low satisfaction scores showed higher churn risk across loyalty groups.
Promotions significantly influenced purchasing behavior among Gold customers.
Certain customer segments contributed disproportionately to electronics revenue.
Spending patterns varied substantially across loyalty categories.
Ranking analysis identified top-value customers within each loyalty segment.

##Data Visualization
A bar chart was created using Matplotlib to compare sales performance between customers who used promotions and those who did not.
The visualization highlighted the impact of promotional campaigns on customer spending.
